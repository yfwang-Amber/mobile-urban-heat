# Mobile Urban Heat Measurements

This folder is the public static website. Publish this folder only.

The parent project folder contains source photos, working material, and a thesis PDF that are not part of the public website. Do not deploy the parent folder as a web root unless those materials have been reviewed and intentionally cleared for public release.

The website is intentionally high level. It avoids raw observations, full transect routes, exact sensor locations, model performance tables, and unpublished scientific claims.

The world map background is a local static asset generated from Natural Earth low-resolution land data; the site shows city-level project markers only.
